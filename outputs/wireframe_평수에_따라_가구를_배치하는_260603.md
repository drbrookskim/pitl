UX Architect로서 'AI 기반 평수 맞춤형 가구 배치 최적화 서비스'를 위한 3가지 UX 전략 시안을 제안합니다.

---

### Type A: 현실/안정형 (Utility-First)
**특징:** 도면 입력과 수치 확인을 최우선으로 하여, 인테리어 초보자가 실수를 줄이고 정확한 배치를 할 수 있도록 돕는 직관적 레이아웃입니다.

```text
┌──────────────────────────────────────────────────────────┐
│ [도면 설정]  가로: 3000mm | 세로: 4000mm | [수정]       │
├──────────────────────────────────────────────────────────┤
│ ┌──────────────────────────┐ ┌──────────────────────────┐ │
│ │                          │ │ [가구 리스트]            │ │
│ │      2D 배치 영역        │ │ 1. 소파 (1800x900) [ADD] │ │
│ │                          │ │ 2. 테이블 (1200x600)[ADD]│ │
│ │    [동선 경고: 450mm]    │ │                          │ │
│ └──────────────────────────┘ └──────────────────────────┘ │
├──────────────────────────────────────────────────────────┤
│ [상태: DRAFT] | [AI 검증: 경고] | [장바구니 담기]        │
└──────────────────────────────────────────────────────────┘
```

---

### Type B: 디자인/감성몰입형 (Visual-First)
**특징:** 시뮬레이션 화면을 전체 화면으로 활용하여 공간감을 극대화합니다. 가구의 심미적 배치를 중시하는 사용자에게 적합하며, 툴바를 하단에 배치하여 몰입도를 높입니다.

```text
┌──────────────────────────────────────────────────────────┐
│ [공간 시뮬레이션]                                [닫기]  │
├──────────────────────────────────────────────────────────┤
│                                                          │
│                                                          │
│                  ( 3D 렌더링 뷰포트 )                    │
│                                                          │
│                                                          │
├──────────────────────────────────────────────────────────┤
│ [ 가구 선택 ] [ AI 자동배치 ] [ 회전 ] [ 치수 보기 ]     │
├──────────────────────────────────────────────────────────┤
│ [ 총 예상 금액: 1,250,000원 ]      [ 일괄 장바구니 담기 ]│
└──────────────────────────────────────────────────────────┘
```

---

### Type C: 미래형 동적 위젯 (AI-Driven)
**특징:** AI가 실시간으로 공간 효율을 분석하여 제안하는 '위젯' 중심의 인터페이스입니다. 사용자는 배치보다는 'AI의 추천'을 선택하고 검증하는 과정에 집중합니다.

```text
┌──────────────────────────────────────────────────────────┐
│ [AI 공간 최적화]                                         │
├──────────┬───────────────────────────────────────────────┤
│ [추천 1] │ ┌──────────┐ [효율: 95%] [동선: 적합] [적용] │
│ [추천 2] │ │ 썸네일   │ [효율: 88%] [동선: 적합] [적용] │
│ [추천 3] │ └──────────┘ [효율: 72%] [동선: 주의] [적용] │
├──────────┴───────────────────────────────────────────────┤
│ [실시간 피드백]                                          │
│ "현재 배치는 동선 효율이 매우 좋습니다. 구매를 추천합니다."│
├──────────────────────────────────────────────────────────┤
│ [배치 완료 및 구매하기]                                  │
└──────────────────────────────────────────────────────────┘
```

---

### UX Architect의 제언
*   **Type A**는 반품률 감소라는 비즈니스 목표에 가장 부합하는 **'안정적 선택'**입니다.
*   **Type B**는 브랜드 경험과 체류 시간 증대에 유리한 **'감성적 선택'**입니다.
*   **Type C**는 AI 기술력을 강조하고 사용자 편의성을 극대화하는 **'혁신적 선택'**입니다.

**추천:** 초기 런칭 시에는 **Type A**의 구조를 기반으로 하되, **Type C**의 AI 추천 위젯을 사이드바 형태로 결합하여 '정확성'과 '편의성'을 동시에 잡는 하이브리드 모델을 권장합니다.
