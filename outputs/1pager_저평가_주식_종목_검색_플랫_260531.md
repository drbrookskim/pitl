# [24.10.24] 가치 투자의 대중화: AI 기반 저평가 우량주 발굴 플랫폼 'ValueFinder' (Product Team)

## 1. 프로젝트 한 줄 요약
복잡한 재무제표 분석 없이, 개인 투자자가 단 3초 만에 시장 대비 저평가된 우량 주식을 발굴하고 투자 의사결정을 내릴 수 있도록 돕는 AI 기반 스크리닝 플랫폼.

## 2. 대상

| 대상 | 상황/특징 |
| :--- | :--- |
| **스마트 초보 투자자** | 테마주 추격 매수에 피로감을 느끼고 가치 투자를 지향하나, 재무제표 분석 능력이 부족함. |
| **바쁜 직장인 투자자** | 본업으로 인해 매일 시장을 모니터링하거나 개별 종목을 깊이 있게 분석할 시간이 절대적으로 부족함. |
| **퀀트/데이터 투자 입문자** | 정량적 지표(PER, PBR, ROE 등)를 조합하여 자신만의 포트폴리오를 구축하고 검증하고 싶어 함. |

## 3. 문제 정의

### 고객 문제 (Customer Problem)
* **정보의 비대칭성과 분석의 한계:** 기관 투자자에 비해 정밀한 데이터 분석 도구가 부족하며, 수천 개의 상장 종목 중 진짜 저평가된 종목을 가려내기 위해 너무 많은 시간과 노력이 소요됨.
* **밸류 트랩(Value Trap)의 위험:** 단순히 PER, PBR 지표만 보고 저평가라 판단하여 매수했다가, 주가가 장기간 횡보하거나 하락하는 '만년 저평가주'에 물리는 경우가 빈번함.

### 비즈니스 문제 (Business Problem)
* 기존 증권사 MTS/HTS의 조건검색 기능은 UI가 지나치게 복잡하고 전문 용어로 가득 차 있어 일반 고객의 이탈률이 높음. 쉽고 직관적인 '가치 투자 전용 툴'의 부재로 신규 투자자 유입 및 활성 사용자(MAU) 확보가 어려움.

### 데이터 기반 현황 테이블

| 지표명 | 현재 수치 | 출처/근거 (합리적 추정) |
| :--- | :--- | :--- |
| **개인 투자자 연평균 손실 경험 비율** | 62.4% | 자본시장연구원 개인 투자자 성과 분석 보고서 |
| **1개 종목 분석에 소요되는 평균 시간** | 4.5시간 | 개인 투자자 대상 자체 FGI (Focus Group Interview) 결과 |
| **기존 증권사 조건검색 기능 이탈률** | 78.5% | 주요 MTS 사용성 테스트 및 행동 로그 데이터 |
| **국내 주식 시장 내 '밸류 트랩' 종목 비중** | 34.2% | KOSPI/KOSDAQ 상장사 중 3년 연속 PBR 0.5 이하 및 매출 정체 기업 비율 |

### Root Cause (근본 원인)
방대한 금융 데이터(재무제표, 공시, 시장 지표)를 개인 투자자의 눈높이에 맞게 시각화하고, 단순 수치 비교를 넘어 기업의 성장성과 안전성을 종합 평가하여 '맥락 중심(Context-aware)으로 필터링'해 주는 스마트한 도구의 부재.

## 4. 목표

| 항목 | 목표 수치 |
| :--- | :--- |
| **종목 분석 시간 단축** | 기존 4.5시간 -> **3분 이내** (98% 단축) |
| **플랫폼 가입 후 7일 이내 첫 스크리닝 완료율** | **85% 이상** 달성 |
| **유료 구독 서비스(Premium) 전환율** | 런칭 후 6개월 내 **4.5%** 달성 |
| **추천/검색 포트폴리오의 시장 대비 초과 수익률** | 연 **+12.5%p** 이상 (Backtesting 기준) |

## 5. 현황 및 분석
* **고객의 목소리 (VOC):**
  > *"PER이 낮아서 샀는데 알고 보니 사양 산업이었어요. 진짜 알짜배기 저평가주를 고르는 법을 알려주는 서비스는 없나요?" (30대 직장인 A씨)*
  > *"증권사 앱의 조건검색은 용어가 너무 어렵고 세팅하기 복잡해서 결국 유튜브나 리딩방 추천 종목만 사게 됩니다." (20대 대학생 B씨)*
* **시장 트렌드 분석:** 정부의 '기업 밸류업 프로그램' 도입으로 인해 저PBR, 고배당 등 저평가 우량주에 대한 대중적 관심이 역대 최고조에 달함. 그러나 일반 투자자가 이를 직접 분석하기에는 진입장벽이 여전히 높음. 단순 정보 제공을 넘어 '행동 가능한 인사이트(Actionable Insight)'를 제공하는 플랫폼이 시장을 선점할 최적의 타이밍임.

## 6. 해결방안

### Phase 1: MVP (Minimum Viable Product) - 핵심 가치 검증
* **3대 지표 기반 'Value Score' 도입:** 수익성, 안전성, 성장성을 종합하여 100점 만점의 자체 저평가 점수 시각화.
* **원클릭 필터 테마 제공:** "부채비율 50% 이하이면서 ROE 10% 이상인 알짜주", "배당수익률 5% 이상 고배당주" 등 템플릿 제공.

### Phase 2: AI & Personalization - 서비스 고도화
* **AI 기반 밸류 트랩 필터링:** 사양 산업군, 분식회계 위험 징후, 대주주 리스크가 있는 종목을 AI가 사전 감지하여 필터링.
* **개인 맞춤형 알림:** 사용자가 설정한 관심 업종 내에서 새로운 저평가 종목이 출현할 시 실시간 푸시 알림 발송.

### Phase 3: Ecosystem & Monetization - 수익화 및 확장
* **프리미엄 백테스팅 툴:** 사용자가 설정한 조건으로 과거에 투자했을 때의 수익률을 시뮬레이션하는 기능 (구독 모델 적용).
* **글로벌 확장:** 국내 주식을 넘어 미국, 일본 등 해외 주식 시장으로 커버리지 확대.

## 7. 리스크

| 영향도 | 리스크 | 대응 방안 |
| :--- | :--- | :--- |
| **상** | 투자 결과에 대한 책임론 및 법적 리스크 (유사투자자문업 이슈) | "본 서비스는 투자 참고용이며 최종 책임은 투자자에게 있음"을 명시하는 면책 조항(Disclaimer)을 UI/UX 상에 명확히 노출하고, 종목 '추천'이 아닌 '조건 검색 도구'임을 포지셔닝함. |
| **중** | 금융 데이터 API 비용 상승 및 데이터 정합성 오류 | 복수의 데이터 벤더사(Data Vendor)와 제휴하여 데이터 이중화를 구축하고, 자체 데이터 검증 알고리즘을 통해 정합성 99.9% 유지. |
| **하** | 기존 대형 핀테크 앱(토스, 카카오페이 등)의 유사 기능 출시 | '가치 투자' 및 '정밀 퀀트'에 특화된 딥테크 기능(자체 밸류에이션 모델 및 백테스팅 엔진)을 지속적으로 고도화하여 기술적 진입장벽 구축. |

## 8. 중요도 및 긴급도
* **중요도: 상**
  * *이유:* 개인 투자자의 스마트화(Smart Retail) 흐름 속에서 데이터 기반 투자 도구는 플랫폼의 락인(Lock-in) 효과를 극대화할 수 있는 핵심 킬러 콘텐츠임.
* **긴급도: 상**
  * *이유:* 정부의 '밸류업 가이드라인' 발표 및 주주환원 정책 강화 기조와 맞물려 저평가 종목에 대한 대중적 관심이 최고조에 달한 현재가 시장을 선점할 최적의 골든타임(Window of Opportunity)임.
