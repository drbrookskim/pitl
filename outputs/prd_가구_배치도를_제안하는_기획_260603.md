# [24.05.22] AI 기반 맞춤형 가구 배치 최적화 솔루션 '스페이스핏(SpaceFit)' PRD

## 0. 문서 개요
본 문서는 사용자의 공간 데이터와 가구 규격을 AI가 분석하여 최적의 배치도를 3D로 자동 생성하는 '스페이스핏' 서비스의 상세 요구사항을 정의합니다. 가구 반품률 감소와 공간 활용 효율 극대화를 목적으로 하며, MVP 단계에서는 공간 치수 입력 기반의 3가지 배치안 자동 생성 기능에 집중합니다.

## 1. 목표 (Goal)
*   **프로젝트 목표:** 수동 배치 대비 90% 이상의 시간 단축 및 데이터 기반의 최적 배치 추천을 통한 고객 만족도 제고.
*   **성공 기준:**
    | 항목 | 목표 수치 |
    | :--- | :--- |
    | 가구 반품률 | 18% → 10% (8%p 감소) |
    | 배치도 생성 시간 | 5분 이내 완료 |
    | 서비스 재방문율 | 3개월 내 35% 달성 |

## 2. KPI 및 측정지표
*   **핵심 KPI:** 가구 반품률, 배치도 생성 완료율, 배치도 기반 구매 전환율.
*   **보조 지표:** 평균 배치도 생성 소요 시간, 3D 뷰어 체류 시간.
*   **모니터링 지표:** API 호출 실패율, 배치 알고리즘 추천 만족도(별점), 가구 DB 매칭 성공률.

## 3. 사용자 및 이해관계자
*   **대상 사용자:** 1인 가구/신혼부부, 이사 예정자, 인테리어 초보자.
*   **제외 대상:** 전문 인테리어 시공이 필요한 대규모 리모델링 고객.
*   **이해관계자:** Product Team(기획/운영), AI 엔지니어링 팀(알고리즘), 프론트엔드 팀(3D 뷰어), 가구 브랜드 파트너사.

## 4. 정책 정의
*   **기본 정책:** 모든 가구 배치는 사용자가 입력한 실측 치수(cm 단위)를 기준으로 하며, 오차 범위 ±2cm를 허용함.
*   **상태값 정의:**
    | 상태값 | 설명 |
    | :--- | :--- |
    | DRAFT | 공간 치수 입력 중 |
    | ANALYZING | AI 배치 알고리즘 연산 중 |
    | COMPLETED | 3가지 배치안 생성 완료 |
*   **예외 정책:**
    *   **결제/API 타임아웃:** 10초 이상 응답 없을 시 '재시도' 버튼 노출 및 서버 로그 기록.
    *   **배치 불가:** 입력된 가구 면적이 공간 면적을 초과할 경우 '배치 불가' 알림 및 가구 제외 제안.

## 5. 서비스 구조 (유저 플로우)
1. **공간 입력:** 평면도 치수 입력 및 문/창문 위치 지정.
2. **가구 선택:** 보유 가구 리스트 선택 및 규격 확인.
3. **AI 연산:** 동선/수납/심미 중심의 3가지 배치안 생성.
4. **결과 확인:** 3D 뷰어를 통한 배치도 확인 및 수정.
5. **구매/저장:** 최종 배치도 저장 또는 연동된 가구 구매.

## 6. 상세 요구사항
| ID | 기능명 | 상세 설명 | 우선순위 |
| :--- | :--- | :--- | :--- |
| REQ-01 | 공간 치수 입력 | 평면도 치수 및 장애물(창문, 문) 입력 UI | P0 |
| REQ-02 | AI 배치 엔진 | 3가지 테마(동선/수납/심미) 배치 알고리즘 | P0 |
| REQ-03 | 3D 뷰어 | 생성된 배치도를 3D로 시각화 및 회전/확대 | P0 |
| REQ-04 | 가구 DB 연동 | 파트너사 가구 규격 데이터 API 연동 | P1 |
| REQ-05 | 결과 공유/저장 | 배치도 이미지 저장 및 URL 공유 | P1 |

*   **운영 요구사항:** 관리자 페이지 내 가구 DB 관리 및 알고리즘 추천 결과 모니터링.
*   **데이터 요구사항:** 사용자 입력 데이터의 익명화 처리 및 학습 데이터 활용 동의 절차.

## 7. UX/UI 고려사항
*   **로딩:** AI 연산 중 '공간을 분석 중입니다'라는 문구와 함께 진행률 바(Progress Bar) 표시.
*   **실패:** API 타임아웃 시 "네트워크가 불안정합니다. 다시 시도해주세요" 팝업 노출.
*   **빈 화면:** 가구 미선택 시 "배치할 가구를 먼저 선택해주세요" 가이드 노출.

## 8. 기술 고려사항
*   **보안:** 사용자 공간 데이터는 암호화하여 저장하며, 개인정보 식별이 가능한 사진 데이터는 즉시 폐기.
*   **성능:** 3D 렌더링 최적화를 위해 WebGL 기반의 경량화된 라이브러리 사용.

## 9. 실험 및 검증
*   **A/B 테스트:** 배치도 추천 테마(3가지)의 클릭률을 분석하여 선호도가 높은 테마를 메인으로 배치.
*   **사용성 테스트(UT):** 2030 사용자 10명을 대상으로 배치도 생성 시간 및 UI 직관성 검증.

## 10. 리스크 및 대응
*   **데이터 정확도:** 실측 가이드 영상 제공 및 오차 범위 보정 알고리즘 강화.
*   **가구 DB 확보:** 주요 브랜드와 API 연동 우선순위 설정 및 파트너십 체결.
*   **학습 곡선:** 직관적인 튜토리얼 영상(30초 내외)을 온보딩 단계에 배치.
