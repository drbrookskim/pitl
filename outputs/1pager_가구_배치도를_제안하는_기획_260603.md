# [24.05.22] AI 기반 맞춤형 가구 배치 최적화 솔루션 '스페이스핏(SpaceFit)' (Product Team)

## 1. 프로젝트 한 줄 요약
사용자의 공간 데이터와 가구 정보를 분석하여, 심미성과 효율성을 극대화한 최적의 가구 배치도를 3D로 자동 생성하는 인테리어 솔루션.

## 2. 대상
| 대상 | 상황/특징 |
| :--- | :--- |
| 1인 가구/신혼부부 | 좁은 공간을 효율적으로 활용하고 싶어 하는 2030 세대 |
| 이사 예정자 | 새로운 공간에 기존 가구가 맞을지 고민하는 사용자 |
| 인테리어 초보자 | 감각적인 배치를 원하지만 공간 지각 능력이 부족한 사용자 |

## 3. 문제 정의
*   **고객 문제:** 가구 구매 후 공간에 맞지 않거나, 배치 실패로 인한 공간 낭비 및 재배치의 번거로움.
*   **비즈니스 문제:** 가구 반품률 증가로 인한 물류 비용 상승 및 고객 경험(CX) 저하.
*   **데이터 기반 현황:**
    | 지표 | 수치 | 비고 |
    | :--- | :--- | :--- |
    | 가구 반품률 | 18% | '사이즈 미스' 및 '배치 부적합' 사유 65% |
    | 공간 활용 만족도 | 42% | 자체 설문조사 결과 |
    | 재배치 고민 시간 | 주당 4.5시간 | 평균적인 가구 배치 고민 시간 |
*   **Root Cause:** 공간 실측 데이터와 가구 규격 간의 시각적 불일치, 그리고 배치에 따른 동선 효율성 계산의 어려움.

## 4. 목표
| 항목 | 목표 수치 |
| :--- | :--- |
| 서비스 이용 후 반품률 감소 | 18% → 10% (8%p 개선) |
| 배치도 생성 완료 시간 | 5분 이내 (기존 수동 배치 대비 90% 단축) |
| 서비스 재방문율(Retention) | 3개월 내 35% 달성 |

## 5. 현황 및 분석
*   **VOC 인용:** "비싼 돈 주고 산 소파가 거실 동선을 다 막아버려서 너무 불편해요. 미리 배치해볼 수 있는 기능이 있었으면 좋겠어요." (20대 여성, 자취생)
*   **분석:** 현재 시장의 인테리어 앱들은 단순 카탈로그 위주이며, 사용자가 직접 수동으로 배치해야 하는 번거로움이 존재함. 자동화된 '최적 배치 추천' 기능이 도입될 경우 강력한 차별점이 될 것임.

## 6. 해결방안
*   **Phase 1 (MVP):** 사용자가 공간 치수 입력 시, 보유 가구 리스트를 기반으로 3가지 최적 배치안(동선 중심, 수납 중심, 심미 중심) 자동 생성.
*   **Phase 2:** AR(증강현실) 기술을 연동하여 실제 공간에 가구가 배치된 모습을 실시간으로 확인하는 기능 추가.
*   **Phase 3:** 가구 추천 알고리즘 고도화(공간에 부족한 가구 추천 및 구매 연동).

## 7. 리스크
| 영향도 | 리스크 | 대응 방안 |
| :--- | :--- | :--- |
| 상 | 데이터 정확도 오류 | 실측 가이드 영상 제공 및 오차 범위 보정 알고리즘 강화 |
| 중 | 가구 DB 확보 지연 | 주요 가구 브랜드와 API 연동 및 파트너십 체결 |
| 하 | 사용자 학습 곡선 | 직관적인 UI/UX 설계 및 튜토리얼 영상 제공 |

## 8. 중요도 및 긴급도
*   **중요도: 상** (고객의 핵심 페인 포인트 해결을 통해 브랜드 충성도 확보 및 반품 비용 절감 가능)
