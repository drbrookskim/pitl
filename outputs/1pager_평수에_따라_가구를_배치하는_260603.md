# [24.05.22] AI 기반 평수 맞춤형 가구 배치 최적화 서비스 (Product Team)

## 1. 프로젝트 한 줄 요약
사용자의 주거 평수와 도면 데이터를 기반으로, 공간 효율을 극대화하고 심미적 만족도를 높이는 'AI 가구 배치 및 큐레이션 서비스'

## 2. 대상
| 대상 | 상황/특징 |
| :--- | :--- |
| 1인 가구/신혼부부 | 좁은 공간을 효율적으로 활용하고 싶은 니즈가 강함 |
| 이사 예정자 | 가구 구매 전 배치 실패(사이즈 미스)를 방지하고 싶은 고객 |
| 인테리어 초보자 | 가구 간 조화와 동선 확보에 어려움을 겪는 고객 |

## 3. 문제 정의
*   **고객 문제:** 가구 구매 후 실제 배치 시 사이즈 불일치, 동선 방해, 공간 낭비 발생으로 인한 반품 및 스트레스 증가.
*   **비즈니스 문제:** 가구 커머스 내 단순 상품 나열 방식은 구매 전환율이 낮으며, 고객의 공간 이해도 부족으로 인한 구매 결정 지연.
*   **데이터 기반 현황:**
    | 지표 | 수치 | 비고 |
    | :--- | :--- | :--- |
    | 가구 반품률 | 18% | 사이즈/배치 불만족이 60% 차지 |
    | 구매 결정 소요 시간 | 14일 | 공간 배치 고민으로 인한 이탈 발생 |
    | 공간 활용 만족도 | 42% | 기존 가구 배치 서비스 부재 |
*   **Root Cause:** 고객이 자신의 주거 공간에 가구가 배치된 모습을 시각적으로 확인하고 검증할 수 있는 '공간 시뮬레이션 도구'의 부재.

## 4. 목표
| 항목 | 목표 수치 |
| :--- | :--- |
| 서비스 이용자 수 | 월간 활성 사용자(MAU) 5만 명 달성 |
| 구매 전환율(CVR) | 기존 대비 15% 상승 |
| 반품률 감소 | 18% → 8% 이하로 개선 |

## 5. 현황 및 분석
*   **VOC 인용:** "가구는 예쁜데 우리 집에 놓으면 좁아 보일까 봐 선뜻 결제를 못 하겠어요.", "줄자로 재는 것도 한계가 있고, 막상 배치하면 동선이 꼬여서 불편합니다."
*   **분석:** 현재 가구 시장은 '상품 중심'에서 '공간 중심'으로 소비 패턴이 변화 중임. AR 기술과 도면 기반 배치 서비스가 결합될 경우 고객의 구매 확신을 크게 높일 수 있음.

## 6. 해결방안
*   **Phase 1 (MVP):** 평수/구조 선택 기반 가구 배치 추천 알고리즘 및 2D 평면도 시뮬레이터 도입.
*   **Phase 2:** AR(증강현실) 기능을 통한 실제 공간 내 가구 배치 미리보기 및 동선 시뮬레이션 기능 추가.
*   **Phase 3:** AI 인테리어 스타일링 추천 및 가구 구매 원스톱 결제 시스템 연동.

## 7. 리스크
| 영향도 | 리스크 | 대응 방안 |
| :--- | :--- | :--- |
| 상 | 도면 데이터 오차 | 사용자 직접 치수 입력 및 보정 가이드 제공 |
| 중 | 가구 DB 확보 지연 | 주요 가구 브랜드와 제휴를 통한 3D 모델링 데이터 확보 |
| 하 | 기술적 진입장벽 | 웹 기반 가벼운 렌더링 엔진 도입으로 접근성 강화 |

## 8. 중요도 및 긴급도
*   **중요도: 상** (가구 커머스의 본질적인 구매 경험을 혁신하고 반품 비용을 획기적으로 줄일 수 있는 핵심 기능임)
*   **긴급도: 상** (경쟁사들이 공간 큐레이션 서비스를 강화하고 있어, 시장 선점을 위해 빠른 MVP 출시가 필수적임)
