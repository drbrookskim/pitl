### 2. 설계 특징 및 UX 전략
*   **화면 목적:** 친숙하고 직관적인 UI 구조를 통해 사용자가 학습 비용 없이 즉시 서비스를 탐색하고 이용할 수 있도록 유도합니다.
*   **정보 구조 (IA):** 상단 GNB ➔ 구독 유도 배너 ➔ 개인화 달성도(리텐션 장치) ➔ 맞춤형 추천 콘텐츠(P0 요건) ➔ 카테고리 탐색 ➔ 탭바 네비게이션.
*   **공통 CTA:** `[ ▶ 10분 루틴 시작하기 ]` (핵심 기능 실행), `[ Premium 구독 시작하기 ]` (구독 전환).
*   **UX 포인트:** 2030 직장인들이 가장 익숙해하는 그리드와 리스트 뷰를 혼합하여 안정감을 주며, 상단에 'FREE 회원' 상태와 '구독 유도'를 명확히 노출하여 비즈니스 목표(구독 전환율 8%)를 직접적으로 타격합니다.

# [Type B] 디자인/감성몰입형 (Emotional & Immersive Layout)

### 1. ASCII 와이어프레임
```text
┌────────────────────────────────────────┐
│ [Menu]          B R E A T H E          │
├────────────────────────────────────────┤
│                                        │
│                                        │
│         "오늘 하루도 수고한 당신,      │
│          오롯이 몸에 집중할 시간"      │
│                                        │
│         ┌────────────────────┐         │
│         │                    │         │
│         │                    │         │
│         │     [  ▶ PLAY  ]   │         │
│         │                    │         │
│         │                    │         │
│         └────────────────────┘         │
│           오늘의 이완: 퇴근길 10분             │
│                                        │
│                o   •   o   o           │
│                                        │
│                                        │
├────────────────────────────────────────┤
│  ┌──────────────────────────────────┐  │
│  │ 7일간 무료로 모든 감성 루틴 만나기  │  │
│  │ [ 무료 체험 시작하기 ]              │  │
│  └──────────────────────────────────┘  │
├────────────────────────────────────────┤
│  [둘러보기]       [나의 공간]       [설정] │
└────────────────────────────────────────┘
```

### 2. 설계 특징 및 UX 전략
*   **화면 목적:** 복잡한 정보 요소를 최소화하고, 감성적인 카피라이팅과 대형 비주얼 카드를 통해 사용자가 앱에 접속하자마자 '힐링'과 '웰니스'의 무드에 몰입하도록 유도합니다.
*   **정보 구조 (IA):** 감성 타이틀 ➔ 대형 단일 추천 카드(Hero 콘텐츠) ➔ 페이지네이션 인디케이터 ➔ 하단 고정형 구독 유도 카드 ➔ 미니멀 탭바.
*   **공통 CTA:** `[  ▶ PLAY  ]` (몰입형 루틴 재생), `[ 무료 체험 시작하기 ]` (감성적 접근의 구독 유도).
*   **UX 포인트:** 텍스트와 복잡한 수치를 배제하고 여백의 미를 강조합니다. 퇴근 후 지친 직장인들이 앱을 켰을 때 피로감을 느끼지 않도록 배려한 레이아웃으로, 브랜드 로열티와 감성적 리텐션을 극대화합니다.

# [Type C] 미래형 동적 위젯 (Dynamic Widget-Based Layout)

### 1. ASCII 와이어프레임
```text
┌────────────────────────────────────────┐
│ [⚡] DYNAMIC DASHBOARD        [Premium]│
├────────────────────────────────────────┤
│ ┌─ [MY BODY STATUS] ─────────────────┐ │
│ │  목/어깨 통증도 : [■■■■■■■□□□] High  │ │
│ │  허리 피로도   : [■■■■□□□□□□] Mid   │ │
│ │  [ 🔍 신체 상태 재진단하기 ]        │ │
│ └────────────────────────────────────┘ │
├────────────────────────────────────────┤
│ ┌─ [QUICK 10M BOOST] ────────────────┐ │
│ │  현재 시간 [14:00] - 오후 잠깨기 ⚡  │ │
│ │  [ ▶ 즉시 스트레칭 시작 ]           │ │
│ └────────────────────────────────────┘ │
├────────────────────────────────────────┤
│ ┌─ [HABIT TRACKER] ──────────────────┐ │
│ │  월[V] 화[V] 수[ ] 목[ ] 금[ ] 토[ ] │ │
│ │  * 2일만 더 채우면 이번 주 미션 완료!│ │
│ └────────────────────────────────────┘ │
├────────────────────────────────────────┤
│ ┌─ [PREMIUM BENEFIT] ────────────────┐ │
│ │  광고 없이 무제한 스트레칭 스트리밍 │ │
│ │  [ 7일 무료 체험 신청 ]             │ │
│ └────────────────────────────────────┘ │
└────────────────────────────────────────┘
```

### 2. 설계 특징 및 UX 전략
*   **화면 목적:** 사용자의 실시간 데이터(시간대, 신체 상태 진단 결과)를 기반으로 개인화된 위젯들을 대시보드 형태로 동적 배치하여, 즉각적인 상호작용을 유도합니다.
*   **정보 구조 (IA):** 신체 상태 모니터링 위젯(REQ-01 기반) ➔ 시간대별 맞춤형 퀵 부스트 위젯(REQ-02 기반) ➔ 습관 트래커 위젯(REQ-04 연동) ➔ 구독 혜택 위젯(REQ-03). *하단 탭바를 없애고 스크롤 가능한 대시보드 형태로 구성.*
*   **공통 CTA:** `[ ▶ 즉시 스트레칭 시작 ]` (상황 맞춤형 원클릭 실행), `[ 7일 무료 체험 신청 ]` (혜택 중심 구독 유도).
*   **UX 포인트:** 테크니컬하고 데이터 중심적인 성향의 사용자에게 적합합니다. 사용자의 현재 시간(예: 오후 2시 사무실)에 맞춰 "오후 잠깨기 스트레칭"을 추천하는 등 동적 위젯 시스템을 통해 개인화 경험을 극대화하고 리텐션(목표 35%)을 확보합니다.

### [요약] UX Architect 제언
*   **Type A (현실/안정형)**는 대중적인 웰니스 앱의 표준으로, **MVP 출시 및 초기 데이터 검증(A/B 테스트)**에 가장 적합합니다.
*   **Type B (디자인/감성형)**는 브랜드 가치와 독점 콘텐츠의 퀄리티를 강조할 때 유리하며, 여성 직장인 타겟층에 소구력이 높습니다.
*   **Type C (미래형 위젯)**는 푸시 알림(REQ-04) 및 사용자 진단 데이터(REQ-01)와 결합했을 때 가장 강력한 **개인화 경험**을 제공하므로, 서비스 고도화 단계에서 도입을 권장합니다.
