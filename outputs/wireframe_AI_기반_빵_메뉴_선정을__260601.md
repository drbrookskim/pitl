안녕하세요. AI 기반 개인 맞춤형 베이커리 큐레이션 서비스 **'빵비게이터(Bread-vigator)'**의 UX Architect 에이전트입니다. 

제시해주신 상세 요건 명세서(PRD)를 바탕으로, 오프라인 매장 고객의 선택 장애를 해결하고 구매 전환율(CVR)을 극대화하기 위한 **핵심 화면인 '추천 결과 화면(Curation Result Screen)'**의 3가지 시안(Type A, B, C)을 설계했습니다.

---

## 0. 공통 화면 목적 및 공통 CTA 정의

*   **화면 목적:** 
    *   고객의 취향 필터(맛, 식감, 건강)와 매장의 실시간 POS 재고 데이터를 결합하여 산출된 **최적의 빵 3종(Top 3)**을 제안합니다.
    *   추천 사유와 실시간 재고 상태를 직관적으로 노출하여 탐색 비용과 의사결정 시간을 단축시킵니다.
*   **공통 CTA (Call To Action):** 
    *   **[할인 쿠폰 받기 / 바코드 생성]**: POS에서 즉시 결제 및 할인이 가능한 바코드를 노출하여 구매 전환율(CVR)과 객단가(AOV) 상승을 유도합니다.
*   **공통 메타 정보:** 
    *   실시간 재고 동기화 상태 표시, 알레르기 유발 물질 주의 문구(법적 면책 고지).

## [Type A] 현실/안정형 (Realistic/Stable)

### 1. 설계 콘셉트 및 특징
*   **타깃:** 디지털 기기 조작이 익숙지 않은 전 연령층 고객, 빠른 구매를 원하는 목적형 구매자.
*   **UX 특징:** 
    *   가장 친숙하고 표준적인 모바일 웹 리스트 뷰 구조를 채택하여 인지 부하를 최소화합니다.
    *   정보 구조(IA)가 위에서 아래로 흐르는 수직형 레이아웃으로, 추천 순위(1~3위)와 상품명, 가격, 재고 상태가 한눈에 들어옵니다.
    *   실시간 재고 상태를 직관적인 텍스트 배지(`[품절임박]`, `[여유]`)로 표현하여 신뢰도를 높입니다.

### 2. ASCII 와이어프레임

│  "오늘의 취향: #담백함 #바삭함 #저당"    │
├──────────────────────────────────────────┤

│    태그: #겉바속촉 #천일염 #버터풍미     │
│    재고: [품절임박 - 2개 남음]           │
├──────────────────────────────────────────┤
│ 2. 크루아상 (Croissant)                  │

│    [ 78% 일치 ]  4,500원                 │
│    태그: #비건 #고식이섬유 #다이어트     │
│    재고: [여유 - 4개 남음]               │
├──────────────────────────────────────────┤
│  [!] 실시간 재고와 일부 차이가 있을      │
│      수 있습니다. (1분 전 동기화됨)      │
├──────────────────────────────────────────┤
│  ┌────────────────────────────────────┐  │
│  │     [ 1,000원 할인 쿠폰 받기 ]     │  │

### 3. 주요 인터랙션 포인트
*   **리스트 카드 클릭:** 각 빵의 상세 성분 및 영양 정보 팝업 노출.
*   **[1,000원 할인 쿠폰 받기] 버튼:** 클릭 시 하단 영역이 확장되며 POS 스캔용 일회용 바코드가 즉시 활성화됨.

## [Type B] 디자인/감성몰입형 (Design/Emotional Immersion)

### 1. 설계 콘셉트 및 특징
*   **타깃:** 트렌드에 민감한 MZ세대, 인스타그램 등 SNS 감성을 선호하는 비주얼 중심 고객.
*   **UX 특징:**
    *   1화면 1카드(Swipe) 방식을 채택하여 추천된 1위 상품에 시각적 극대화 및 몰입감을 부여합니다.
    *   건조한 스펙 나열 대신 "오늘, 당신의 지친 하루를 채워줄..."과 같은 감성적인 마이크로카피와 맛/식감의 시각적 스케일(게이지 바)을 제공합니다.
    *   여백의 미를 살리고 텍스트를 최소화하여 브랜드 아이덴티티를 강조합니다.

### 2. ASCII 와이어프레임

```text
┌──────────────────────────────────────────┐
│  B R E A D - V I G A T O R               │
├──────────────────────────────────────────┤
│  "오늘, 당신의 지친 하루를 채워줄        │
│   따뜻한 한 조각을 찾았습니다."          │
├──────────────────────────────────────────┤
│  ┌────────────────────────────────────┐  │
│  │  [ 이미지: 노릇노릇한 소금빵 ]     │  │
│  │                                    │  │
│  │  소금빵 (Salt Bread)               │  │
│  │  "겉은 바삭, 속은 쫄깃한 버터의    │  │
│  │   풍미와 짭조름한 소금의 만남"     │  │
│  │                                    │  │
│  │  단맛  [■□□□□]  식감 [바삭함]     │  │
│  │  재고  "방금 오븐에서 나왔어요!"   │  │
│  └────────────────────────────────────┘  │
│             <  1  /  3  >                │
├──────────────────────────────────────────┤
│  ┌────────────────────────────────────┐  │
│  │     [ 이 감성 그대로 쿠폰 받기 ]   │  │
│  └────────────────────────────────────┘  │
│  * 알레르기 주의: 우유, 밀 함유          │
└──────────────────────────────────────────┘
```

### 3. 주요 인터랙션 포인트
*   **좌우 스와이프 (`< 1 / 3 >`):** 손가락 제스처 또는 화살표 클릭을 통해 2위, 3위 추천 빵 카드로 부드러운 트랜지션 효과와 함께 전환.
*   **감성 게이지:** 맛의 강도를 시각적인 인디케이터(`[■□□□□]`)로 표현하여 직관적인 맛 유추 가능.

## [Type C] 미래형 동적 위젯 (Future Dynamic Widget)

### 1. 설계 콘셉트 및 특징
*   **타깃:** 테크 친화적인 얼리어답터, 실시간 데이터와 효율성을 중시하는 스마트 컨슈머.
*   **UX 특징:**
    *   마치 SF 영화의 대시보드나 실시간 주식 차트를 보는 듯한 위젯 기반의 레이아웃입니다.
    *   AI 엔진의 매칭 연산 과정과 POS 재고 동기화 상태(`[POS_SYNC: 12s ago]`)를 실시간으로 시각화하여 기술적 신뢰감을 극대화합니다.
    *   매칭률 레이더 차트(막대형 그래프로 대체)와 실시간 재고 경고 위젯이 유기적으로 작동합니다.

### 2. ASCII 와이어프레임

```text
┌──────────────────────────────────────────┐
│ [AI_ENGINE_v1.2]        [SYS_STATUS: OK] │
├──────────────────────────────────────────┤
│ >> USER_PREFERENCE_MATCHING_COMPLETE     │
│ >> REALTIME_POS_INVENTORY_FILTER_ACTIVE  │
├──────────────────────────────────────────┤
│ [MATCH RADAR WIDGET]                     │
│  ┌────────────────────────────────────┐  │
│  │  소금빵     [■■■■■■■■■□] 98%  │  │
│  │  크루아상   [■■■■■■■■□□] 85%  │  │
│  │  통밀식빵   [■■■■■■■□□□] 78%  │  │
│  └────────────────────────────────────┘  │
├──────────────────────────────────────────┤
│ [LIVE INVENTORY STATUS]                  │
│  ┌────────────────────────────────────┐  │
│  │  - 소금빵:   [!! 품절임박] 2개 남음 │  │
│  │  - 크루아상: [NORMAL]      5개 남음 │  │
│  │  - 통밀식빵: [NORMAL]      4개 남음 │  │
│  └────────────────────────────────────┘  │
├──────────────────────────────────────────┤
│  ┌────────────────────────────────────┐  │
│  │ >> [ GENERATE_DISCOUNT_BARCODE ]   │  │
│  └────────────────────────────────────┘  │
│ [POS_SYNC: 12s ago]   [SAFETY_STOCK: ON] │
└──────────────────────────────────────────┘
```

### 3. 주요 인터랙션 포인트
*   **실시간 카운트다운:** POS 재고 변동 시 위젯 내 숫자가 깜빡이며 실시간으로 갱신되는 시각 효과 제공.
*   **[GENERATE_DISCOUNT_BARCODE] 버튼:** 클릭 시 사이버네틱한 사운드/애니메이션과 함께 바코드 팝업이 로드됨.

## 4. 시안별 비교 및 추천 매트릭스

| 비교 항목 | Type A (현실/안정형) | Type B (디자인/감성형) | Type C (미래형 위젯) |
| :--- | :--- | :--- | :--- |
| **주요 타깃** | 전 연령층, 대중적 고객 | 2030 여성, 감성 소비층 | 테크 얼리어답터, 남성/젊은 층 |
| **정보 가독성** | **최상** (익숙한 리스트 구조) | 중 (순차적 정보 탐색 필요) | 상 (데이터 시각화 우수) |
| **감성적 몰입도**| 하 (기능적 정보 전달 중심) | **최상** (스토리텔링 및 비주얼) | 중 (기계적이고 차가운 느낌) |
| **의사결정 속도**| 빠름 (한눈에 비교 가능) | 보통 (스와이프 탐색 시간 소요) | **매우 빠름** (재고 긴박감 조성) |
| **구현 난이도** | 낮음 (표준 UI 컴포넌트) | 보통 (스와이프 및 모션 그래픽) | 높음 (실시간 웹소켓/폴링 연동) |

### UX Architect 제안:
*   **Phase 1 (MVP 단계)**에서는 사용성 검증과 빠른 구현을 위해 **[Type A: 현실/안정형]**을 기본 레이아웃으로 채택하는 것을 권장합니다.
*   이후 브랜드 인지도가 확보되고 감성 마케팅이 필요한 시점에 **[Type B]**를, 스마트 스토어 콘셉트의 플래그십 매장 오픈 시에는 **[Type C]**를 적용하여 A/B 테스트를 진행하는 로드맵을 제안합니다.
